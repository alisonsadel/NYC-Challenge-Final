About This Project


Machine Learning - NYC Real Estate Valuations

Overview

Group Members: Stephen Brescher, Shadee Tabasi, Alison Sadel, Manny Mejia
Our team wanted to create an interactive dashboard featuring NYC real estate prices broken down by zip code, property type, # of bedrooms and # of baths. A key component of our dashboard would utilize a Machine Learning Model to spit out an approximate property price based on features such as property type, street tree density, street tree health, brownfield proximity, walkability, proximity to subway stations and local school data.
